組合語言

雷諾瓦：「痛苦會過去，美會留下來。」
是的，組合語言帶來的磨難皆會隨著學期的落幕而過去，而你我淒美如落花的死狀則會永遠留存下來。

1. register

32bits：EAX, EBX, ECX, EDX
16bits：AX, BX, CX, DX
8 bits： A, B, C, D
…

所有A共用同一塊記憶體，8bit高低位有分al,ah。
一般mov指令會把高位之處填0。

2. data

1. 進位表示法：h=16, d=10, b=2 / ex: 32d, 6Ah, 1101b
2. ASCII字元：1byte=8bit

  • 8bit: BYTE, SBYTE
  • 16bit: WORD, SWORD
  • 32bit: DWORD, SDWORD
  • 64bit: QWORD

3. flag

  1. Zero flag (ZF): 若指令後的目標為0時，ZF=1。
  2. Sign flag (SF): 若指令後的目標為負時，SF=1。
  3. Carry flag (CF): 應進位、無號數溢位時，CF=1。
  4. Overflow flag (OF): 有號數溢位時，OF=1。

mov指令不會影響任何flag

4. progeaming

  • library linking: INCLUDE Irvine32.inc。
  • main
.data
...

.code
main PROC
    ...
main ENDP
END main ;開始執行的位置
  • procedure
MyProcedure    PROC  USES esi ecx    ;所需參數傳遞

MyProcedure    PROC
    ...
    ret
MyProcedure    ENDP

隨後再以call呼叫程序，使用時原先的「下個指令」會暫存於EIP。

進入程序後需自行於程序內保留外面需要用到的暫存器於stack中。

  • 參數傳遞：pushAD、popAD。

5. initialize

string1 BYTE "ASSEMBLY",0

little endian: 低位放在位置較前處。

6. mnemonic

  • $ 代表現在位置
  • call DumpRegs display register
  • mov 移動
mov      al, var1
mov      al, [var1]      ;調整型別
mov      EAX, [var2+4]   ;位置加減以bit計算,若要改位置需加[]
movzx    ax, bl          ;補0
movsx    ax, bl          ;補符號數
  • xchg ax, bx 交換，最多只能有一個來自mem。
  • INC a, b加1、DEC減1、ADD加、SUB減、NEG reg正負相反。

地址位置相關

  • OFFSET回傳資料的位置，返還64bit。
mov	ESI, OFFSET [list1]
call	WriteHex                ;回傳list1地址

.data
str1	BYTE "abcdefg",0
.code
mov	EDX, OFFSET str1
call	WriteString             ;印出字串str1
  • PTR取其位置內的數字，並轉換成指定型別後移動到distance。可通用於mov EAX, [A]，但目的地EAX與[A]必須同型別。
.data
A    DWORD    10h

.code
mov  AX, WORD ptr [A]    ;轉換成WORD
mov  EAX, DWORD ptr [A]  ;轉換成DWORD
  • TYPE回傳型別所佔的byte數量。LENGTHOF回傳變數中element的數量；SIZEOF回傳變數所佔的byte數量，即TYPE*LENGTHOF。
mov    EAX, TYPE var1
mov    EAX, LENGTHOF var1
mov    EAX, SIZEOF var1
  • LABLE 略
  • 間接地址的使用
.data
val1    BYTE    10, 20, 30

.code
mov     esi, OFFSET val1     ;OFFSET把val1的地址放入sei
mov     al, [esi]            ;esi內放地址,間接地址必打[]

[]表示其內資料視為地址，即C當中的*

  • 若要用mov ax, [array+4]要用4，即byte數量、index*type，而非array index數量。
  • 標籤(lable)，區域標籤如L1:，全域標籤如L1::。

跳躍與迴圈

  • jmp L1 無條件跳躍。
  • loop L1 會自動更新ecx-1直到0停止。
mov    ecx, 5    ;這裡放要loop的次數
L1:
    ...
    loop    L1

mov    ecx, 5
L1:
    mov    count, ecx    ;把外層的index目前數量存入記憶體
    mov    ecx, 10       ;放入內層要用的ecx次數
    L2:
        ...
        loop    L2
    mov    ecx, count    ;把外層index目前數量從記憶體取回
    loop   L1
  • loopNZ 跳出的條件為ECX>0且ZF=0。

堆疊

  • SS式stack的位置，ESP為stack pointer。
  • push放入stack；pop取出stack中最上層者。
push    ecx
...
pop     ecx
mov     eax, [ESP]    ;亦可存到stack中最上層資料
  • pushFD、popFD讓flag存入EFLAGS。
  • pushAD、popAD把32bit暫存器存入。
  • pushA、popA把16bit暫存器存入。
  • call DumpMem

stack中第一格進去的位置(return address)會放入ESP。

  • call ArrayFill把預定array的位置全都填上隨機16bit數值。
ArrayFill PROC
    push ebp
    mov  ebp,esp
    pushad
    mov  esi,[ebp+12]
    mov  ecx,[ebp+8]

EBP用來表示base address，通常於程序中是不會被改變的，是基數。程序結束(return)時需要置換回原本的數值(利用push和pop)。

  • ret程序結束後return。ret n代表返還後將stack中的n bit資料丟棄。
;法1
push    val1
call    procese
add     esp, 8

;法2
push    ebp
mov     ebp, esp
...
pop     ebp
ret     8
  • 建立區域變數
MySub    PROC
    push    ebp
    mov     ebp,esp
    sub     esp,8                   ;留空間
    mov     DWORD PTR [ebp-4],10    ;存入,建立變數初始化
    mov     DWORD PTR [ebp-8],20    ;存入,建立變數初始化
  • LEA回傳stack中資料的參數或區域變數地址(OFFSET只能用在常數)
CopyString    PROC, count:DWORD
LOCAL         temp[20]:BYTE
lea           edi, count
lea           edi, temp

邏輯運算

  • and、or、xor、not…
mov    al, 'A'
and    al, 11011111b    ;直接將and結果存於and

跳躍

  • jmp 無條件跳躍。
  • test 略
  • cmp 比較兩者，結果將影響下一行跳躍指令。
  • 若某flag=1則跳：JB,JZ,JS；若某flag=0則跳：JNZ,JECXZ。
  • 基於cmp指令的比較：
    JA >
    JAE >=
    JB <
    JBE <=
    … signed略
cmp    EAX, 5
JA     L1:
L1:
    ...

型別轉換

  • CWB convert BYTE(EAX) to WORD。
  • CWD convert WORD(EAX) to DWORD。
mov    eax, DWORDval
cwb

整數運算

  • SHL邏輯向左移動、原先最左位移動到CF；SHR邏輯向右移動且左邊補0、原先最右位移動到CF。
mov    dl, 5
shl    dl, 2    ;相當於乘以4
  • SAL、SAR算術位移、保留正負號且移動到CF中。
  • ROL、ROR左右旋轉，即所有bit都往旁移動一格、最旁者移動回第一格 & CF 中。
  • RCL、RCR經過carry的左右旋轉。右旋轉中CF當最右格進行旋轉；左旋轉中CF當最左格，即n+1格的旋轉。
  • SHLD、SHRDregister移動至另一個register長度。略。
  • WVAL左移4bit
  • MUL用於乘法，EAX上超過32bit的高位會存於EDX，若有32bit進位(EDX非0)則 CF=1。
mov    ax, val1
mul    val2
  • IMUL用於有號數乘法，若有進位則 OF=1。
  • DIV用於除法前，需先將DX(或EDX)清除，且DIV只會有一個參數，即除數，被除數與商皆在AX(或EAX)。
mov    edx, 0
mov    eax, 8003h
mov    ecx, 100h
div    ecx            ;EDX:EAX=EAX/ECX
  • IDIV有號數除法在做之前先須將目標(distance)做sign-extended、把AX延伸入DX或EAX延伸入EDX。
mov    eax, -48
cdq
mov    ebx, 5
idiv   ebx
  • ADC連同進位的加法，用於EDX:EAX。
  • SBB連同借位的減法，用於EDX:EAX。
mov    edx, 0            ;先清除EDX
mov    eax, FFFFFFFF
add    eax, 1            ;先做低位加法
adc    edx, 0            ;再做高位加法、看CF決定是否再+1

ASCII相關

  • BCD將數字轉ASCII (1 -> ‘1’)
  • AAS調整SUB、SBB結果
  • AAM調整MUL結果
  • AAD調整BCD結果
  • DAA
  • DAS
